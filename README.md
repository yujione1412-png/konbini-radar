# コンビニレーダー 🟢 v1.3.0

現在地・指定した場所・移動ルートの沿線から、まわりのコンビニをかんたんに探せるスマホ向けWebアプリです。
**`index.html` 1ファイルだけで動きます**（ビルド不要・完全無料・カード登録不要）。

## できること

- 📍 **現在地のまわりを探す** … 探す範囲は 200m〜3000m のスライダーで調整。スライダーを動かすと表示中の検索を自動でかけ直す（地図やルートはそのまま）
- 🎯 **指定した場所で探す** … 地図を動かして画面中心の十字を合わせ「十字の位置で探す」。長押しで◎ピンも置ける（ルートの出発地になる）
- 🛣️ **ルート沿いを探す** … 「ルート沿い」を押して案内に従うだけ。出発地は「現在地から出発」ワンタップでもOK
- 🗺️ **店までのルート表示** … 徒歩／車を切り替え可能。Googleマップで開くこともできる（ナビは自動開始しません）
- 🕘 **履歴** … 表示したルートを端末内に保存し、あとからワンタップで再表示

## 使い方（デプロイ）

1. このリポジトリをGitHubに置く（`index.html` があればOK）
2. [Vercel](https://vercel.com) でリポジトリを Import（Framework Preset は **Other**、ビルド設定は不要）
3. 以降は GitHub の `index.html` を差し替えて Commit するだけで、同じURLが自動更新されます

> 位置情報は **https のページでしか動きません**。パソコンでファイルを直接開くと現在地が取れないので、Vercelに置いたURLをスマホで開いてください。

## 使っている無料サービス（すべてカード登録不要）

| 用途 | サービス |
|---|---|
| 地図表示 | Leaflet 1.9.4 + OpenStreetMap |
| コンビニ検索（主） | YOLP Yahoo!ローカルサーチAPI（1日5万件まで無料） |
| コンビニ検索（予備） | Overpass API（OpenStreetMap） |
| ルート検索 | OSRM（FOSSGIS公開サーバー・徒歩／車） |
| 外部地図 | Google Maps URLs（リンクで開くだけ・キー不要） |

## 知っておいてほしい限界

- YOLPでも店の取りこぼしはゼロになりません（0件・失敗のときはGoogle Mapを開くボタンが出ます）
- OSRM / Overpass の公開サーバーは混みあうと失敗することがあります（時間をおいて再試行してください)
- 履歴は端末内（localStorage）のみに保存され、他の端末とは共有されません

## 開発を続けたいとき（AIへの引き継ぎ）

`index.html` の冒頭コメントが「セーブデータ」です。ファイル全体をAIのチャットに貼り付けて
「このファイルの続きから追加して」と伝えれば、バージョン履歴・仕様を引き継いで続きを作れます。
詳しい仕様は [docs/仕様書.md](docs/仕様書.md) を参照してください。
